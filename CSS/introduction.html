CSS stands for cascading style sheet which is used to design the web pages.

There are three types of CSS.
        i> Inline/Embedded CSS-worst practise
        ii> Internal CSS-written inside same web page in head section
        iii> External CSS-written in external file making .css file and connect them with HTML pages.-good practise.
